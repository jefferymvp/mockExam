# MockExamApp

这是一个基于 **UniApp** 框架开发的模拟考试应用，旨在提供轻量级的离线考试练习功能。

## 技术栈 (Tech Stack)

*   **核心框架**: [UniApp](https://uniapp.dcloud.io/) (基于 Vue.js 2.x)
*   **开发语言**: JavaScript (ES6+)
*   **样式语言**: CSS / SCSS
*   **路由管理**: UniApp 内置路由
*   **数据管理**: 本地静态数据 (无后端交互)

## 功能模块 (Functionality)

### 1. 首页 (Home)
*   **题型选择**: 支持单选题、多选、判断题或混合模式。
*   **题量设置**: 可通过滑块自定义题目数量，系统会根据题库库存自动调整最大值。
*   **开始考试**: 校验配置后跳转至考试页面。

### 2. 考试页 (Exam)
*   **随机出题**: 根据所选配置从题库中随机抽取题目。
*   **答题交互**: 支持单选、多选和判断题的交互操作。
*   **即时反馈**: 提交答案后立即判断对错，并显示正确答案。
*   **结果统计**: 考试结束后展示正确率和答对题数。

## 项目结构 (Project Structure)

*   `pages/`
    *   `index/`: 首页，负责考试配置。
    *   `exam/`: 考试页，负责答题和结果展示。
*   `utils/`
    *   `questionBank.js`: 模拟题库数据及随机抽题逻辑。
*   `main.js`: Vue 初始化入口。
*   `pages.json`: 路由及页面配置。
*   `manifest.json`: 应用配置信息。
